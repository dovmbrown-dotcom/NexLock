Three cabinet-locking concepts explored, compared, and refined for everyday family use.

Ideate Solution

To address cabinet-access risks involving medicines, cleaning products, detergents, and sharp household items, we developed and evaluated three locking concepts. Each concept was judged on practical use in real family homes: practicality, compactness, ease of installation, ease of daily use, cost efficiency, and alignment with user feedback.

Practicality Compactness Ease of Use Installation Cost User Feedback

Solution 1 — Mechanical Rack & Pinion Lock

Concept

Overview: A motorized rack-and-pinion locking system that uses claw hooks to secure cabinet doors and block access to dangerous cabinet contents.

Strengths

  • Strong mechanical hold
  • Durable structure
  • Clear locking mechanism

Limitations

  • Bulky internal system
  • Higher production cost
  • More complex installation

Solution 2 — Electromagnet Mount System

Concept

Overview: Two compact internal electromagnets automatically secure cabinet doors when closed, helping prevent children from accessing medicines, cleaners, and other hazardous items stored inside.

How It Works

Auto-align
Magnets align when the cabinet shuts
Reinforce hold
Alignment studs increase holding force
One-touch release
External mounted button releases magnets
Open alert
Buzzer alerts if the cabinet is left open after use

Strengths

  • Compact and hidden
  • Automatically locks
  • One-touch unlock
  • Lower-cost components
  • Simple installation
  • Button cannot be lost

Limitations

  • Requires a power source
  • Magnet strength must be verified for reliability

Solution 3 — Biometric Deadbolt System

Concept

Overview: A fingerprint-activated deadbolt system designed to unlock cabinet access only for authorized users.

Strengths

  • Personalized access
  • High-tech appeal
  • Secure authentication

Limitations

  • Expensive components
  • Over-engineered for cabinets
  • More complicated installation

Modern security with a smooth, easy look.
